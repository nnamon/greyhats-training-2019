# 12 April

## Overview

This session aims at covering the following topics:

1. Data representation
2. Executables and Programs
3. Assembly
4. Registers
5. The Stack
6. C Programs
7. String Representation
8. Buffer Overflows
9. deBrujin Generators
10. GEF
11. Shellcode
12. pwntools
13. srand\(\)
14. Intro to Mitigations
